This Module Generator package provides various generators for Caffeinated Modules including CRUD, API, Controller, Model, Migration, View .

## Requirements
    Laravel >= 5.7
    PHP >= 7.1.3

## Installation
```
composer require developeroncall/modulegenerator 
```
